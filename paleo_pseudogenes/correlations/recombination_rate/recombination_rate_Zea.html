Required libraries

library(MareyMap)
library(corrr)

Background

Here, we estimate the recombination rate along the genome (in bins) using the Marey Map approach. Next, we then assess whether there is a (negative) correlation between recombination rate and pseudogene number.

paper with Zea mays dataset: https://doi.org/10.1038/s41467-018-04562-5

**MAYBE TRY OTHER DATASET: https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0028334#s5

Reference of recombination landscape inference in flowering plants: https://doi.org/10.1371/journal.pgen.1010141

Aim

Find relationship between pseudogenes and recombination rate

Run MareyMap

startMareyMapGUI()

We use the same strategy as Brazier and Glemin, 2022:

  • Two-degree polynomial loess regression

  • Span = 0.2

  • Negative recombination rates set to 0

Bin per 500 kb windows and plot result

# Split recombination rate data in bins of 500 kb
df$bin <- cut_width(df$phys, width = 500000)

# Get average local recombination rate by taking the average of the loess estimate
# for the markers in that bin
mean_rec <- df %>% group_by(map, bin) %>%
  summarize(mean_GR_rate = mean(loess, na.rm = TRUE))
## `summarise()` has grouped output by 'map'. You can override using the `.groups`

cor_table <- bins_wide %>%
  correlate(method = "pearson") %>%
  select(-c(bin, mean_GR_rate)) %>%
  filter(term == "mean_GR_rate")
## Non-numeric variables removed from input: `map`
## Correlation computed with
## • Method: 'pearson'
## • Missing treated using: 'pairwise.complete.obs'
calc_pval <- function(vec_a, vec_b, meth){
  cor.test(vec_a, vec_b, method = meth)$p.value
}

pval_table <- bins_wide %>%
  select(-c(bin, map)) %>%
  colpair_map(calc_pval, meth = "pearson") %>%
  select(-mean_GR_rate) %>%
  filter(term == "mean_GR_rate")

cor_table_spear <- bins_wide %>%
